Bienvenido al OVI del curso Diseño de sitios Web a continuación encontrará información básica sobre el manejo del aplicativo git a través de contenidos multimedia que darán paso a paso una guía desde cero para trabajar con este programa y diseñar páginas web.

Inicio

Objetivo General

Dar a conocer y profundizar en los fundamentos y usos de Git y Github, HTML5 y CSS3 para diseñar y crear sitios web.


Objetivo específico 1: Mostrar por medio de información los fundamentos del control de versiones (Git) y su manejo a través de este sitio Web.

Objetivo específico 2: Reconocer e identificar la nueva tendencia del HTML5 que representa la estructura y el contenido de los sitios web y con ello lograr el desarrollo de los mismos.

Objetivo específico 3: Brindar conocimientos sobre el manejo de CSS3, para dar un mejor aspecto en el diseño de sitios Web.
